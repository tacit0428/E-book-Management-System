## **:blue_book::books: 电子书后台管理系统**

- 应用Vue+Node.js+Express+MySQL技术，基于Element-UI组件库搭建后台管理系统
- 通过Node实现了EPUB电子书的解析、新增、编辑、删除和查找
- 基于MySQL进行电子书数据存储，并将资源上传到Nginx静态资源服务器
- 利用JWT实现了token的生成、校验和路由过滤，从而实现用户登录功能
